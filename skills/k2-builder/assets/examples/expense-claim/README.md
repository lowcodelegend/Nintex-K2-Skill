# EXP.Expense Claims

A disposable complete-solution example for the K2 master-detail contract. It creates a SQL-backed expense-claim header, line-item table, lookup table, summary view, generated SmartObjects, a capture/item master View, an editable-list detail View, and a tabbed Form whose Form-level rules persist and reload the relationship.

Deploy data first, then forms. The fixture is designed to be removed after live verification.
