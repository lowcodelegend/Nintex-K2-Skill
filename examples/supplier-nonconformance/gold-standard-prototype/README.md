# Gold-standard case UX prototype

This platform-neutral prototype is the visual and interaction target for the reusable case-management experience. It intentionally exceeds the current SmartForms implementation before any platform compromises are made.

Open `index.html` in a modern browser. No build step or service is required. The prototype includes:

- role-oriented command centre with prioritised operational intelligence;
- full work queue and case search;
- five-step resumable initiation journey;
- persistent case context, lifecycle, decision brief, evidence provenance and governed next actions;
- report catalogue and global command/search palette;
- desktop, tablet and mobile layouts; keyboard focus, reduced-motion handling and semantic status labels.

The data is fictional. AI content is explicitly advisory and requires accountable human review.
